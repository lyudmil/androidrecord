AndroidRecord is a free, open-source ORM library for Android inspired by ActiveRecord. It makes it ridiculously easy to write database backed applications for Android phones.

Features

Zero configuration

Just drop AndroidRecord into your project and implement your models. AndroidRecord will create the database schema for you. No config files.

Effortless querying

AndroidRecord provides a number of convenient ways to query your database. Saving, updating, deleting, and finding records is a cinch.

Entity relationships

If your business logic calls for relationships between your models, AndroidRecord handles that for you.
