AndroidRecord is a free, open-source ORM library for Android inspired by ActiveRecord. It makes it ridiculously easy to write database backed applications for Android phones.

Features

Effortless querying

AndroidRecord provides a number of convenient ways to query your database. Save, update, delete, and find records without writing tedious boilerplate code.

Zero configuration

Just drop AndroidRecord into your project, implement your models, and it magically sets up your database. Absolutely no configuration involved.

Entity relationships

If your business logic calls for relationships between models, AndroidRecord handles that for you.
